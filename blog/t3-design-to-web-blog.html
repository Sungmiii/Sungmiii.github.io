What a responsive site is, and why responsiveness is important?

When you open a same website between your laptop, desktop and a mobile phone, you are able to find the screen looks different. This is the responsive site which is automatically fitted an any device screen.

What mobile first design is, and why it's important?

Nowadays most people are accessing the internet via their mobile phone. If we make a website designed first for desktop, there are lots of interaction such as hover but mobile phone is not showing a hover interaction. So, we need to think the mobile first design to do not work too much for nothing.

What frameworks are, and their pros and cons?

Framework is a standard of layout and concepts for building website. Thier pros that we can build the website faster and safely, cons that we are learning the code just copying and pasting.

What a wireframe is and why we use it?

A wireframe is the foundation for designing the website, we can't just writing down code in html and css stylesheet. We need a plan and wireframe is the first thing what we want to looking our website to be.
